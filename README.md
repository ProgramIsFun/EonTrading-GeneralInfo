# EonTrading-GeneralInfo


| Contract Type    | Execution Timing                            | Has Strike/Exercise Price? | Has Contract Price?    | What It Means                     |
|------------------|---------------------------------------------|----------------------------|------------------------|-----------------------------------|
| European Option  | Only on expiry date                         | Yes                        | No                     | Can be exercised only at expiry   |
| American Option  | Anytime before and including expiry date     | Yes                        | No                     | Can be exercised at any time      |
| Warrants (HK)    | Usually only on expiry date (European style)| Yes                        | No                     | Most are European style           |
| Futures          | Settled/exercised on expiry date            | No                         | Yes                    | Obliged to settle on expiry date  |
